# astra-theme (published build)

This repository holds the **build artifacts** for the
[astra-theme](https://github.com/LightconeResearch/astra-theme) MyST web theme.
It is generated by CI — do not edit by hand. Open issues and PRs against the
source repo.

Use it in a MyST project:

```yaml
site:
  template: https://github.com/Eiffl/astra-theme
```

Published from `v0.0.5`.
